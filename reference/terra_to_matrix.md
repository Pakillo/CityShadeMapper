# Convert a SpatRaster to matrix

Convert a SpatRaster to matrix

## Usage

``` r
terra_to_matrix(ras = NULL)
```

## Arguments

- ras:

  A
  [`terra::SpatRaster()`](https://rspatial.github.io/terra/reference/SpatRaster-class.html)
  object, or a path (character) to a raster file.

## Value

A matrix
