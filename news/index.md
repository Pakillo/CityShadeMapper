# Changelog

## CityShadeMapper 0.0.2

- Added new logical argument `low.veg.as.ground` to
  [`rasterize_lidar_cover_class()`](https://pakillo.github.io/CityShadeMapper/reference/rasterize_lidar_cover_class.md)
  to decide if low vegetation should be considered ground or mixed with
  medium-high vegetation
  ([\#21](https://github.com/Pakillo/CityShadeMapper/issues/21)).
